Introduction¶

Scientific Question: Are the differences in the infectivity of Zaire Ebolavirus and Influenza type A explained by differences in the structures of their surface proteins, spike glycoprotein and hemagglutinin respectively.¶

Influenza Type A binds to cell surface receptors by its The Zaire strain of Ebolavirus (ZEBOV) binds to cells through its glycoprotein, which is the only protein expressed on the surface of the cell. It is also essential in the evasion of the immune system. Influenza Type A is known for having a very high transmissibility

Scientific Hypothesis: If Zaire Ebolavirus and Influenza type A's infectivities are explained by their different surface proteins, glycoprotein and hemagglutinin respectively, which they use to bind to cells, then we should expect to see very little similarity between their protein sequences.¶

I used pairwise sequence alignment to detect regions of similarity between the amino acid sequences of the two surface proteins of Zaire Ebolavirus and displayed the results in a heat map. I got the sequences for this from Uniprot I also conducted structural homology on the two proteins through Swissmodel with the amino acid sequences and the results are in the form of 3D protein measurements.

Loading in Packages¶

Bio: Biopython is a collection of non-commercial python tools. These tools are specifically for computational biology and bioinformatics. It is used to represent biological sequence data and sequence annotations and used to read various file types. For this project I used biopython for multiple sequence alignment. For more information on Biopython, see https://en.wikipedia.org/wiki/Biopython

Seaborn: Seaborn is a data visualization library. It was based on the other package matplotlib. With it, user's are able to create detailed and colorful graphics. It is able to make various kinds of different graphics. With it I created a heatmap to display my analysis data. For more information on Seaborn, see https://seaborn.pydata.org/

Nglview is a package that is used to view molecular structures and trajectories. It allows the user to interact with a rendered 3D image of a molecule. One is able to completely rotate it in any direction and zoom in/out. With this I was able to render a 3D image of a modeled molecule for structural homology analysis. With this you can completely rotate it and zoom around. For more information of Nglview, see https://package.wiki/nglview

IPython: IPython is a command shell for interactive computing in python. It can be used for detailed object introspection. It also has various supplemental functions with it. For this I used it to load a png into my python. This is so that it was embedded in the html and so that it didn't show up in a pop up window when running the code. For more information on ipython, see https://pypi.org/project/ipython/

Performing Bioinformatics Analysis¶

The fasta files ("InfluenzaA_AAs.fasta", and "ZEBOV_AAs.fasta") uploaded contain the sequence of amino acids of the proteins being investigated. These were used for pairwise sequence alignment to find regions in the sequences that aligned with each other

For structural homology modeling, I used Swissmodel to create a model that showed the highest area of similarity between the 2 proteins. I input the amino acid sequences of the proteins for this and it created a pdb file that I input below.
